# 課程連結

- 適用年級：小四至小六（教師示範可延伸至較低年級）
- 科目：STEAM、電腦科、跨科科學探究
- 建議堂數：2 節
- 學習重點：感測器輸入、事件、狀態、訊息發布／訂閱、可靠控制與故障診斷
- 先備知識：FutureLite 基本接線、Soil／LED 本地測試、簡單條件及循環
- 學習顯證：學生能解釋 telemetry、command、ACK、timeout 及裝置離線的分別
- 常見誤解：把 raw 當百分比；以「已發送」當作實體完成；把 self-test 當作裝置在線
- 差異化：初階只觀察狀態與圖表；進階比較 command ID、latency 及壞訊息處理
- 評估：以雙向驗收表及口頭解釋通訊路徑作形成性評估

本工具不代表教育局指定教材；課程定位屬校本 STEAM／電腦科實作安排。
